API Documentation

Integrate Xoiq AI into your applications

REST API & WebSocket Integration

Our comprehensive API allows you to integrate Xoiq AI's accent-aware conversational capabilities into any application. Choose between REST endpoints for simple integrations or WebSocket connections for real-time processing.

Base URL

https://api.xoiq.ai/v1
POST /api/v1/speech/analyze

Analyze speech input and detect accent

Request Example
{
  "audio_data": "base64-encoded-audio",
  "format": "wav",
  "sample_rate": 16000
}
Response Example
{
  "accent": "scottish",
  "confidence": 0.96,
  "text": "Hello, I need help with my account",
  "processing_time_ms": 85
}
POST /api/v1/speech/harmonize

Apply accent harmonization to improve clarity

Request Example
{
  "text": "Original transcribed text",
  "detected_accent": "geordie",
  "target_clarity": "standard"
}
GET /api/v1/accents/supported

Get list of supported accents

Response Example
{
  "accents": ["scottish", "welsh", "northern-irish", "geordie", "yorkshire", "cornish", "brummie", "scouse", "devon", "norfolk", "cockney", "rp", "west-country", "midlands"],
  "total_count": 14
}
WebSocket wss://api.xoiq.ai/v1/realtime

Real-time speech processing with live accent detection

Connection Example
const ws = new WebSocket('wss://api.xoiq.ai/v1/realtime');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Accent detected:', data.accent);
};

SDK Examples

// JavaScript SDK Example
import { XoiqAI } from 'xoiqai-sdk';

const client = new XoiqAI({
  apiKey: 'your-api-key',
  region: 'uk'
});

// Analyze speech for accent
const result = await client.speech.analyze({
  audioData: audioBlob,
  format: 'wav'
});

console.log('Detected accent:', result.accent);
console.log('Confidence:', result.confidence);
# Python SDK Example
from xoiqai import XoiqAI

client = XoiqAI(
    api_key='your-api-key',
    region='uk'
)

# Analyze speech for accent
result = client.speech.analyze(
    audio_data=audio_bytes,
    format='wav'
)

print(f"Detected accent: {result.accent}")
print(f"Confidence: {result.confidence}")
// Java SDK Example
import com.xoiq.ai.XoiqAI;
import com.xoiq.ai.models.SpeechAnalysis;

XoiqAI client = new XoiqAI.Builder()
    .apiKey("your-api-key")
    .region("uk")
    .build();

// Analyze speech for accent
SpeechAnalysis result = client.speech()
    .analyze(audioData, "wav")
    .execute();

System.out.println("Detected accent: " + result.getAccent());
System.out.println("Confidence: " + result.getConfidence());
